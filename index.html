Agent Infrastructure

idle

Quality gate plugin for Claude Code. Blocks exit until work passes review by an independent agent.

Agent works → tries to exit → Stop hook → alice reviewed? → block/allow

LLMs struggle to evaluate their own outputs. A model asked to verify its work tends to confirm rather than critique. idle applies multi-agent debate: rather than prompting agents to review themselves, it blocks exit until alice (an adversarial reviewer) explicitly approves.

Skills: reviewing, researching, issue-tracking, technical-writing, bib-managing

Shell AGPL-3.0 GitHub

zawinski

Email for agents. Async messaging with identity and git context.

Agents need to communicate asynchronously—one kicks off a task, another picks it up later, a third reviews the result. zawinski stores messages in an append-only JSONL log that merges cleanly in git. Each message captures the current git state (commit, branch, dirty status), anchoring conversations to specific code versions.

jwz post tasks -m "Review the auth module" --model claude-3-opus
jwz reply 01HQ -m "Analysis complete. Found 3 issues."
jwz search "auth"

Named after Jamie Zawinski, in reference to Zawinski's Law: "Every program attempts to expand until it can read mail."

Zig MIT JSONL + SQLite FTS5 GitHub

tissue

Issue tracking for agents. Non-interactive, git-native, machine-first.

Most issue trackers are built for humans clicking through web UIs. Agents can't answer interactive prompts or parse HTML. tissue is non-interactive by design: JSON on stdout, errors on stderr, meaningful exit codes. The underlying storage is append-only JSONL that merges cleanly in git.

id=$(tissue new "Fix flaky tests" -p 1 --quiet)
tissue comment "$id" -m "Root cause: race condition"
tissue status "$id" closed
tissue ready  # Show unblocked work

Inspired by Steve Yegge's Beads memory system for coding agents.

Zig MIT JSONL + SQLite FTS5 GitHub

Runtime Libraries

termcat

Cell-based terminal I/O. Diff rendering, cross-platform, Unicode-aware.

Terminal libraries fall into two camps: simple ones like termbox that work everywhere but lack features, and complex ones like notcurses that do everything but are hard to port. termcat aims for the middle ground.

The core abstraction is a cell grid. You write characters to cells, and termcat diffs against the previous frame to emit only the escape sequences that changed. Unicode handling follows terminal expectations: wide characters occupy two cells, combining marks attach to their base.

var term = try termcat.Terminal.init(allocator, .{});
defer term.deinit();

term.draw(0, 0, "Hello", .green, .default, .{ .bold = true });
try term.present();
Zig MIT POSIX + Windows 10+ GitHub

laser

BEAM runtime in Zig. Run Erlang/Elixir bytecode, bundle standalone executables.

The BEAM is excellent for long-running services, but its startup time makes it impractical for CLI tools and serverless functions. An Erlang release takes hundreds of milliseconds just to boot the VM.

laser is a minimal BEAM implementation in Zig that starts in under a millisecond. It compiles to a single static binary with no runtime dependencies, and you can cross-compile to any target Zig supports.

laser run hello.beam main 0
laser info module.beam
Zig MIT OTP 28 bytecode Design Phase GitHub

glypha

Constraint-based graphics DSL for programmatic technical diagrams.

A Lisp-like language for defining technical and scientific figures with S-expression syntax, constraint-based layout, and structured diagnostic API for agent queries.

(define c1 (circle :id "set-a" :cx 100 :cy 100 :r 80
                   :fill "#ff000044" :stroke "#ff0000"))
(define c2 (circle :id "set-b" :cx 160 :cy 100 :r 80
                   :fill "#0000ff44" :stroke "#0000ff"))
(group :id "venn" c1 c2)
glypha eval diagram.gly > output.svg
glypha query diagram.gly '(all-collisions)'
Zig MIT SVG output GitHub

Elixir Ecosystem

absynthe

Syndicated Actor Model for Elixir. Implementation of Tony Garnock-Jones' SAM with Preserves.

Traditional message-passing actor systems rely on ephemeral messages. Absynthe uses assertions—persistent facts shared in dataspaces. Actors observe patterns and react when facts appear or disappear. All changes commit atomically, providing consistent snapshots.

Actor → asserts facts → Dataspace → notifies observers → reactions

loupe

Self-hosted semantic search. Keep your data on your infrastructure.

Services like Pinecone and Weaviate offer convenience, but they require data to leave your servers. For organizations handling confidential materials, this presents unacceptable security risks.

loupe runs semantic search on PostgreSQL with pgvector. Documents are embedded via your chosen API, stored locally, and searched with vector similarity. Terraform templates provision AWS infrastructure.

EndpointMethodDescription
/searchPOSTQuery with vector similarity
/indexPOSTAdd documents to index
/healthGETService health check
/statsGETIndex statistics

bibval

Citation validator. Checks BibTeX against academic databases.

BibTeX files accumulate errors. You copy a citation from Google Scholar, but the year is wrong. You cite a preprint that's since been published. bibval catches these by checking entries against the source of truth.

Quick Start

Install the Plugin Suite

# Install idle (includes zawinski, tissue, and all dependencies)
curl -fsSL https://github.com/evil-mind-evil-sword/idle/releases/latest/download/install.sh | sh

# Add emes marketplace to Claude Code
claude /plugin marketplace add evil-mind-evil-sword/marketplace

# Install idle plugin
claude /plugin install idle@emes

Enable Review Mode

Add #idle:on to the start of your prompt to enable adversarial review. Use #idle:off to disable.
